

### Git Commands 

##### git stash

```bash

 # To check stash list
   git stash list
   
# To stash the changes
   git stash 
   
 # To updated stash with messages
   git stash push -m "any messages"
   
 # To apply the most recent stash 
   git stash apply
  
  # To apply a specific stash 
   git stash apply [index]

  # To apply and remove the most recent stash 
   git stash pop 
   
  # To apply and remove a specific stash 
   git stash pop [index]
   
  # To remove most recent stash 
   git stash drop 

  # To remove a specific stash 
   git stash drop [index]

  # To clear stash list
   git stash clear

  


```

### File-Folder Creation


```bash

 # To make Directory
  mkdir [directoryname]
 # To remove directory 
  rmdir [directoryname]
 # To rename directory 
  mv [old_name] [new_name]
  # To create a new file 
  touch [filename.txt]
 # To Rename File 
  mv [filename.txt] [newname.txt]
 # To Delete file 
  rm [filename.xt]
```
